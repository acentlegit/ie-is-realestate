# Private Banking Theme (from header palette)

Classic “Private Banking” – safe, premium, high-end without being flashy.

## Palette

| Role | Hex | Use |
|------|-----|-----|
| **Primary Navy** | `#003152` | Primary text, primary buttons, headings |
| **Muted Steel Blue** | `#51697A` | Secondary text, muted labels |
| **Soft Blue-Gray** | `#BECDD9` | Soft background areas |
| **Light Gray** | `#DDE1E3` | Border lines, dividers |
| **Near White** | `#F4FCFD` | Page background |
| **Surface / Card** | `#FFFFFF` | Cards, panels, inputs |
| **Accent (Gold)** | `#C8A24A` | Accent buttons, highlights, active states |

## CSS variables (living-space.css)

- `--bg` → #F4FCFD  
- `--card` → #FFFFFF  
- `--stroke` → #DDE1E3  
- `--text` → #003152  
- `--muted` → #51697A  
- `--accent` → #C8A24A  
- `--good` / `--warn` / `--bad` → status green / amber / red  

## Best for

Wealth, finance, corporate, private client vibes.
