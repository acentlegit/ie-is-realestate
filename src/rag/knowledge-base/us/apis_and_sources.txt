**Socrata API Documentation:**
- URL: https://dev.socrata.com/

**Example: Cook County (IL) Property Sales Dataset**
- URL: https://datacatalog.cookcountyil.gov/resource/wvhk-k5uv.json?$limit=100

## 2. Data Access Methods

### Socrata (SODA) API Usage
- Many US counties use Socrata platform
- RESTful API with JSON responses
- Supports filtering, aggregation, and pagination
- Rate limits vary by portal

### Census API Usage
- Free, no authentication required for basic data
- Supports multiple data sets (ACS, Decennial Census)
- Geographic filters by state, county, tract, block group

### Commercial APIs
- ATTOM: Requires API key, subscription-based
- Regrid: API key required, usage-based pricing
- First American: Enterprise access required
